Der Anwendungs-Assistent hat folgende Ressourcen erstellt:

Test.rc
    Hierbei handelt es sich um eine Auflistung aller Ressourcen von Microsoft Windows, die
    vom Programm beim Kompilieren für eine Plattform verwendet werden, die Standard-Windows
    CE Shell verwendet. Sie enthält die Symbole, Bitmaps und Cursor, die im
    Unterverzeichnis RES gespeichert sind. Diese Datei lässt sich direkt in Microsoft
    Visual C++ bearbeiten. Wenn die RC-Datei beibehalten wird, werden die Definitionen im Datenabschnitt
    als Hexadezimalversion des numerischen Werts beibehalten, 
    auf den sie festgelegt sind, und nicht als Anzeigename der Definition.

Test.rc2
    Diese Datei enthält Ressourcen, die nicht mit Microsoft 
    Visual C++ bearbeitet werden.  Fügen Sie in diese Datei alle Ressourcen ein,
    die nicht mit dem Ressourcen-Editor bearbeitet werden können.

Resource.h
    Dies ist die Standardheaderdatei, die neue Ressourcen-IDs definiert.
    Die Datei wird mit Microsoft Visual C++ gelesen und aktualisiert.

Testppc.rc
    Hierbei handelt es sich um eine Auflistung aller Ressourcen von Microsoft Windows, die
    vom Projekt beim Kompilieren für die Pocket PC 2003-Plattform oder eine Plattform verwendet werden, 
    die dasselbe Benutzeroberflächenmodell unterstützt. Sie enthält die Symbole, Bitmaps, 
    und Cursor, die im Unterverzeichnis RES gespeichert sind.  Diese Datei lässt sich 
    direkt in Microsoft Visual C++ bearbeiten. Wenn die RC-Datei beibehalten wird,
    werden die Definitionen im Datenabschnitt als Hexadezimalversion
    des numerischen Werts beibehalten, auf den sie festgelegt sind, und nicht als Anzeigename
    der Definition.

Testppc.rc2
    Diese Datei enthält Ressourcen, die nicht mit Microsoft 
    Visual C++ bearbeitet werden.  Fügen Sie in diese Datei alle Ressourcen ein,
    die nicht mit dem Ressourcen-Editor bearbeitet werden können.

Resourceppc.h
    Dies ist die Standardheaderdatei, die neue Ressourcen-IDs definiert.

Test.ico
